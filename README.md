# fe_co - Facturación electrónica Colombia (DIAN) para Odoo 18 Community

Repositorio de addons para:
- Emisión directa a DIAN: Factura de Venta, Nota Crédito, Nota Débito, Exportación, Documento Soporte
- RADIAN (eventos): emisión y consulta/sincronización
- Contingencia (activación manual)

## Addons incluidos (starter)
- `l10n_co_edi_base`
- `l10n_co_edi_ubl21`
- `l10n_co_edi_signature`
- `l10n_co_edi_dian_direct`
- `l10n_co_edi_radian`
